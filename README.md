# UI Chatbot

Web app stack: React + TypeScript + Vite

## Setup app locally

- cd into project ```cd lseg-chatbot```
- Install dependencies: ```npm i```
- Run: ```npm run dev````

- The application will run on http://localhost:5173/lseg-chatbot/

- Tests: ```npm test```

## Deployment

The application is deployed and accessible on:
